Marketing website for **Ember**, a speech-coaching app where you practice speaking 2 minutes a day, get AI-powered feedback from animal coaches, and track your progress as your campfire grows.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Create a `.env.local` file:

```
CONVERTKIT_API_KEY=your_kit_api_key
```

This powers the waitlist signup via the [Kit v4 API](https://developers.kit.com).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run start` | Serve production build |
| `npm run lint` | Run ESLint |
